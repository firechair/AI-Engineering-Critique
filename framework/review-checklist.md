# AI Output Review Checklist

Use this checklist before accepting any AI-generated artifact.

## Code Generation
- [ ] **Functionality**: Does it run?
- [ ] **Safety**: Any security risks (SQL injection, etc.)?
- [ ] **Style**: Does it match the existing codebase?
- [ ] **Hallucination**: Are imports and libraries real?
- [ ] **Complexity**: Is there a simpler way?

## Content Generation
- [ ] **Fact Check**: Verify all numbers, dates, and names.
- [ ] **Tone Check**: Is it consistent?
- [ ] **Banned Words**: Are there unwanted phrases?
- [ ] **Formatting**: Are headers and lists consistent?

## Architecture / Design
- [ ] **Feasibility**: Can this actually be built?
- [ ] **Scalability**: Will it break under load?
- [ ] **Cost**: Is it too expensive to run?
